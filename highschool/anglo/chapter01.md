# Numeric expressions

## Series 1
```
4 + 5 * 8 = ?
```

```
7 + 2 * 6 = ?
```

```
3 + 6 * 1 = ?
```

```
11 + 4 * 3 = ?
```

### Resolution
```
4 + (5 * 8)
= 4 + 40
= 44
```

```
7 + (2 * 6)
= 7 + 12
= 19
```

```
3 + (6 * 1)
= 3 + 6
= 9
```

```
11 + (4 * 3)
= 11 + 12
= 23
```

## Series 2

```
31 - 15 * 2 = ?
```

```
19 + 18 : 2 = ?
```

```
27 - 32 : 4 = ?
```

```
36 - 28 : 7 = ?
```

### Resolution
```
31 - (15 * 2)
= 31 - 30
= 1
```

```
19 + (18 : 2)
= 19 + 9
= 28
```

```
27 - (32 : 4)
= 27 - 8
= 19
```

```
36 - (28 : 7)
= 36 - 4
= 32
```

## Series 4

```
5 * (7 + 4) + 3 * 2 = ?
```

```
7 * (8 : 4) - 4 * 3 = ?
```

```
(8 - 3) * 4 + 18 : (5 - 2) = ?
```

```
(3 + 9) : 3 - 4 : (7 - 5) = ?
```

### Resolution
```
5 * (7 + 4) + 3 * 2
= (5 * 11) + (3 * 2)
= 55 + 6
= 61
```

```
7 * (8 : 4) - 4 * 3
= (7 * 2) - (4 * 3)
= 14 - 12
= 2
```

```
(8 - 3) * 4 + 18 : (5 - 2)
= (5 * 4) + (18 : 3)
= 20 + 6
= 26
```

```
(3 + 9) : 3 - 4 : (7 - 5)
= (12 : 3) - (4 : 2)
= 4 - 2
= 2
```
