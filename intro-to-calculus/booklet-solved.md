### Question No. 4

Determine the interval (s) constituted by the real numbers `x` that satisfy the given inequality:

a) |x - 3| ≤ 1  
b) |2x + 1| > 3  

#### Solution for question No. 4

a)
Developing solution:
```
= |x - 3| ≤ 1
= (x - 3) + 3 ≤ 1 + 3
= x - 3 + 3 ≤ 4
= x ≤ 4
or
= (x - 3) + 3 ≥ -1 + 3
= x - 3 + 3 ≥ 2
= x ≥ 2
```

Presenting solution:
```
S = {x ∈ ℝ | 2 ≤ x ≤ 4}
or
x ∈ [2, 4]
```

b)
Developing solution:
```
= |2x + 1| > 3
= (2x + 1) + (-1) < -3 + (-1)
= 2x + 1 - 1 < -3 - 1
= 2x < -4
= x < -2
or
= (2x + 1) + (-1) > 3 + (-1)
= 2x + 1 - 1 > 3 - 1
= 2x > 2
= x > 1
```

Presenting solution:
```
S = {x ∈ ℝ | x < -2 or x > 1}
or
x ∈ (-∞, -2) ⋃ (1, ∞)
```

j) ³√81 * (6 ^ ²⁄₃) ÷ 2 ^ ⁴⁄₃

Developing solution:
