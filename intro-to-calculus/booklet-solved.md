### Question No. 1

Use inequalities to describe the given range:

a)
```
-----[-----------)-----
    -2           3
```

b)
```
-----(----------------------->
    -4           
```

#### Solution for question no. 1

a)
```
  -2 ≤ x < 3
```

b)
```
  x > -4
```

### Question No. 3

Determine the distance between two real numbers:

a) 0 and 3  
b) -5 and -2  

#### Solution for question no. 3

a)
```
  3 - 0
  = 3 total distance.
```

b)
```
  (-2) - (-5)
  = 3 total distance
```
